## OctopusAI
OctopusAI contains many AI agents.

## Install

**To install from PyPI(once available), run:**

```bash
   pip install octopusai
```

## Setup

1. **Set your OpenAI API key**:

```bash
   export OPENAI_API_KEY='your-openai-api-key'
```

## Commands

To view all available commands, run:
```bash
   octopusai --help
```
